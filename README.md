# El Alba 2026 — Santísimas Cruces de Chancayllo

Invitación interactiva para el **Alba 2026** de la Festividad de las Santísimas Cruces de Chancayllo.

Organizado por **Juan Carlos La Rosa Andrade** junto a familiares, amigos y acompañantes.

## 📂 Archivos

| Archivo | Descripción |
|---------|-------------|
| `index.html` | Página principal con toda la invitación |
| `style.css` | Estilos y diseño visual |
| `script.js` | Interactividad (cruces, galería, cuenta regresiva) |
| `images_data.js` | Imágenes de la festividad en base64 |

## 🚀 Cómo usar en GitHub Pages

1. Sube todos los archivos a un repositorio de GitHub
2. Ve a **Settings → Pages**
3. En **Source** selecciona `main` y `/ (root)`
4. ¡Listo! Tu invitación estará en `https://tu-usuario.github.io/nombre-repo`

## ✨ Secciones

- 📜 **Historia** — Línea del tiempo desde 1562 hasta 2026
- 📆 **Cronograma** — Los 4 días completos de la festividad
- ⏰ **Programa** — Horarios del Alba con detalle
- ✝ **Las 13 Cruces** — Toca cada cruz para conocer su historia
- 📷 **Galería** — Fotos reales de la festividad
- ⏳ **Cuenta regresiva** — Tiempo real hasta el 19 de junio
